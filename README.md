# 🍷 Wine Quality Prediction using Machine Learning

This is my **first Machine Learning project** where I built a model to predict wine 
quality based on physicochemical properties of wine.  
The model achieves **85% accuracy** using proper preprocessing, outlier detection, and evaluation techniques.

---

## 📌 Problem Statement
Wine quality evaluation is traditionally performed by human experts, which can be subjective and time-consuming.  
This project aims to automate wine quality prediction using Machine Learning algorithms based on chemical attributes of wine.

---

## 📊 Dataset
The dataset contains physicochemical properties of wine such as:

- Volatile Acidity
- Alcohol
- pH
- Sulphates

**Target Variable:** Wine Quality

---

## ⚙️ Machine Learning Workflow

- Data Cleaning & Preprocessing
- Outlier Detection using **Local Outlier Factor (LOF)**
- Stratified Train-Test Split
- Feature Scaling (StandardScaler)
- Model Training
- Model Evaluation

---

## 📈 Model Performance

- ✅ Accuracy: **85%**
- ✅ Precision Score
- ✅ Recall Score
- ✅ F1 Score
- ✅ Confusion Matrix Analysis
